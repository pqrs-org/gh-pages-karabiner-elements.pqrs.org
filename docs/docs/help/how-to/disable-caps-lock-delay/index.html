How to disable caps lock delay

Enable Disable caps_lock delay setting from Complex Modification.

  1. Import Change caps_lock key setting.
  2. Enable Disable caps_lock delay.

Note

Actually, this setting does not disable the caps lock delay of macOS.

This setting puts a wait between caps lock key down and key up event in order to ensure macOS recognize the caps lock event.
